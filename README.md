### 🏍️ BikeTribe — *In Development*
 
A community platform for motorcycle riders. Real-time ride tracking, gang/group system with roles, live route sharing, achievements tied to actual kms ridden, and a full social layer.
 
**Stack:** React · FastAPI · WebSockets · PostgreSQL · Maps API
 
**What's interesting about it:**
- Gang system with structured roles (leader, co-leader, members)
- Live ride-along interface with support crew coordination
- Achievement engine based on group distance, not just activity
- Built to scale from 10 to 10,000 riders without rewrites
